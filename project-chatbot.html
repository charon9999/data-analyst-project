Product Substitution Chatbot

🤖 AI-Powered Matching 📈 Cost Savings Analysis 🔧 FastAPI, Gemini, ChromaDB
PythonFastAPI Gemini AIChromaDB RAGVector Search

How It Works

Enter a product you currently purchase and the chatbot will find the best substitute products from our catalog using AI-powered semantic search (ChromaDB vector embeddings) combined with Gemini LLM reasoning. It calculates quantity conversions, compares pricing, and shows potential cost savings.

Tech Stack Summary

  • Data Collection: Python scrapers (BeautifulSoup, Selenium, Requests) → MongoDB
  • Data Processing: Pandas ETL, Excel join for negotiated prices → MySQL
  • Vector Store: ChromaDB with all-MiniLM-L6-v2 embeddings (121K+ products indexed)
  • AI Engine: Gemini LLM for intelligent ranking, quantity conversion, and comparison
  • API: FastAPI backend served via Docker + Caddy (HTTPS)
  • Frontend: Vanilla JS chatbot with localStorage history persistence
