Project Overview

An AI-powered product substitution tool that helps procurement teams find equivalent or better alternatives to products they currently purchase. Enter a product you buy, and the system uses semantic search across 121K+ catalog items, then Gemini LLM ranks the top matches by fit, calculates quantity conversions, and estimates cost savings.

Deployment & Tech Stack

Infrastructure

  • Hosting: Linode cloud server running the API as a Docker container
  • API Framework: FastAPI serves the /categories and /substitute endpoints, communicating between the frontend (GitHub Pages) and the backend services
  • Containerization: Docker Compose orchestrates the FastAPI app, ensuring consistent deployment and easy scaling
  • HTTPS: Caddy reverse proxy with auto-provisioned Let’s Encrypt SSL certificate (via DuckDNS)

Data Pipeline

  • Data Collection: Python scrapers (BeautifulSoup, Selenium, Requests) extract product data from ODP & Staples → stored in MongoDB
  • Data Processing: Pandas ETL cleans, transforms, and joins scraped data with pre-negotiated contract prices from Excel → loaded into MySQL
  • Vector Store: Qdrant indexes 120K+ active products using BAAI/bge-small-en-v1.5 embeddings via fastembed for semantic similarity search

AI & Search

  • Semantic Search: User query is embedded and matched against Qdrant to retrieve the top 50 most similar products filtered by supercategory and category
  • LLM Ranking: Gemini 2.5 Flash evaluates the 50 candidates, selects the top 5 best substitutes based on product fit, specifications, and use case
  • Savings Engine: Automatically calculates quantity conversion (e.g., pack sizes) and compares unit/total pricing to estimate cost savings
