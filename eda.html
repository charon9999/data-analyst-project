Built as a Jupyter Notebook on Kaggle, demonstrating proficiency in cloud notebook environments. The analysis uncovers revenue patterns, customer behavior, product category performance, geographic distribution, return rates, and data quality issues.

Outlier Detection (IQR Method)

Python
for col in numerical_cols:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR

    outliers = df[(df[col] < lower_bound) | (df[col] > upper_bound)]
    if len(outliers) > 0:
        print(f"{col}: {len(outliers)} outliers")

# Found outliers in 7 columns:
# Unit Price: 8 (2.32%), Qty: 47 (13.62%)
# Calc Total: 12, Total: 12, Discounted Total: 16

Key Findings

Revenue Insights

  • Total Revenue: ₹9,07,708 across 345 orders (Avg Order Value: ₹2,631)
  • Outerwear is the top category at ₹3.19L (35% of revenue), followed by Dresses at ₹2.20L
  • Outerwear has the highest AOV at ₹4,687 — nearly 4× that of Accessories (₹1,292)

Customer & Channel Behavior

  • 78% of customers are New (269), only 22% are Returning (76)
  • Website leads order source with 120 orders, followed by Mobile App (116) and Instagram Shop (109)
  • Credit Card is the top payment method (88 orders), followed by UPI (82) and COD (62)

Return Analysis

  • 52.17% return rate — 180 out of 345 orders returned (⚠ high!)
  • Size Issue is the #1 return reason (67 returns, 37%), followed by Quality Defect (32)
  • 173 refunds processed, 7 still pending

Geographic Distribution

  • Maharashtra (₹1.28L) and Gujarat (₹1.24L) are the top revenue states
  • Mumbai (27 orders) and Delhi (26 orders) are the top cities
  • Delhi warehouse fulfills the most orders (132), followed by Mumbai (121)

Data Quality

  • 12 columns have missing values — Comments column is 98.26% empty
  • Discount-related fields have 59.42% null values (only 140 orders used discount codes)
  • 7 columns have outliers via IQR method; Qty column has highest outlier rate (13.62%)

Operational Metrics

  • Delivery rate: 94.78% (327 delivered, 17 RTO, 1 cancelled)
  • Average delivery time: 14.3 days (range: 3–46 days)
  • Average return window: 26.1 days
