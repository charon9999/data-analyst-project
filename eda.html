Missing Values Analysis

Python
def missing_values_analysis(df):
    missing = pd.DataFrame({
        'Column': df.columns,
        'Missing_Count': df.isnull().sum(),
        'Missing_Percentage': (df.isnull().sum() / len(df) * 100).round(2)
    })
    missing = missing[missing['Missing_Count'] > 0]
    missing = missing.sort_values('Missing_Count', ascending=False)

    # Visualize missing values
    plt.figure(figsize=(10, 6))
    plt.barh(missing['Column'], missing['Missing_Percentage'])
    plt.xlabel('Missing Percentage (%)')
    plt.title('Missing Values by Column')
    plt.tight_layout()
    plt.show()

Outlier Detection (IQR Method)

Python
def outlier_detection(df):
    numerical_cols = df.select_dtypes(include=[np.number]).columns

    for col in numerical_cols:
        Q1 = df[col].quantile(0.25)
        Q3 = df[col].quantile(0.75)
        IQR = Q3 - Q1
        lower = Q1 - 1.5 * IQR
        upper = Q3 + 1.5 * IQR

        outliers = df[(df[col] < lower) | (df[col] > upper)]
        if len(outliers) > 0:
            print(f"{col}: {len(outliers)} outliers detected")

Key Findings

Revenue Insights

  • Electronics and Footwear are the highest revenue product categories
  • Top 3 states (Maharashtra, Karnataka, Delhi) contribute over 60% of total revenue
  • Monthly trend shows seasonal fluctuations with peak in Q1

Customer Behavior

  • Returning customers make up 60% of total customers
  • UPI is the most popular payment method, followed by Credit Card
  • Website and App channels contribute roughly equally to order volume

Data Quality

  • Missing values found primarily in delivery and return-related columns
  • Outliers detected in order amounts using IQR method
  • Comprehensive quality report generated with completeness scores per column

Return Analysis

  • Size Issue is the most common return reason
  • Return rate calculated across all order statuses
  • Majority of refunds have been processed successfully
