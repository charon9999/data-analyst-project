Web Scraper — Staples

Extracts product data from Staples Advantage by hitting their internal JSON API directly. No HTML parsing or browser automation needed — pure HTTP requests for fast, lightweight scraping.

⚙ Internal API Access 🔧 Python, Requests, JSON 📦 JSON + Image Storage
Python Requests JSON API No Selenium REST API
⬇ Download staples_scraper.py

How It Works

1

Extract Product ID

Parses the numeric product ID from the Staples URL using regex. Constructs the internal API endpoint: /ele-lpd/api/sba-sku/product_{id}.

2

Hit Internal API

Makes a direct HTTP GET to the Staples Advantage backend API. Handles JSON-based 301 redirects (Staples returns the redirect path inside the response body). No browser automation needed.

3

Parse JSON Response

Extracts product name, pricing, specifications, description bullets, reviews (rating + count), image URLs, and availability from the structured SBASkuState JSON response.

4

Download & Store

Downloads product images directly via HTTP from Staples' image CDN. Saves everything as structured JSON with local image references.

API-First Approach

            
Staples Advantage exposes an internal ele-lpd API that returns structured JSON. By calling it directly, we skip HTML parsing and browser automation entirely — making scraping 5-10x faster than Selenium-based approaches.

          

          

            
JSON Redirect Handling

            
Staples returns 301 redirects inside the JSON body (not as HTTP headers). The scraper detects "status": 301 in the response and follows the "path" field to construct the canonical API URL.

          

          

            
Registry Pattern

            
Each supplier implements a common interface. The registry maps supplier keys (e.g. "odp", "staples") to their scraper class, enabling plug-and-play supplier additions without modifying core logic.

  2from urllib.parse import urljoin, urlparse

  5class StaplesAdvantageScraperr:
  6    def __init__(self, output_dir="staples_products"):
  7        self.output_dir = output_dir
  8        self.images_dir = os.path.join(output_dir, "images")
  9        os.makedirs(self.images_dir, exist_ok=True)
 10        self.session = requests.Session()
 11        self.headers = {
 12            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; ...',
 13            'Accept': 'application/json, text/plain, */*',
 14            'Referer': 'https://www.staplesadvantage.com/',
 15        }
 16
 17    def scrape_product(self, product_url):
 18        """Main scraping method — uses Staples internal API."""
 19
 20        # Extract product ID from URL
 21        match = re.search(r'product[_/](\d+)', product_url)
 22        product_id = match.group(1)
 23
 24        # Construct API URL and fetch
 25        api_url = f"https://www.staplesadvantage.com/ele-lpd/api/sba-sku/product_{product_id}"
 26        response = self.session.get(api_url, headers=self.headers, timeout=30)
 27
 28        # Handle 301 redirect (Staples returns redirect path in JSON body)
 29        if response.status_code == 301 or 'RedirectHandler' in response.text:
 30            redirect_data = response.json()
 31            if redirect_data.get('status') == 301:
 32                redirect_path = redirect_data.get('path', '')
 33                api_url = f"https://...sba-sku{redirect_path}"
 34                response = self.session.get(api_url, ...)
 35
 36        data = response.json()
 37        return self.extract_product_data(data)
 38
 39    def extract_product_data(self, api_response):
 40        """Extract and structure product data from API response."""
 41        sku_state = api_response.get('SBASkuState', {})
 42        items = sku_state.get('skuData', {}).get('items', [])
 43        item = items[0]
 44        product = item.get('product', {})
 45        price_data = item.get('price', {})
 46
 47        # Extract specifications from productAttributes
 48        specifications = {}
 49        for spec in product.get('productAttributes', []):
 50            specifications[spec['name']] = {
 51                'value': spec['value'],
 52                'description': spec.get('dscr', '')
 53            }
 54
 55        # Get pricing from nested price structure
 56        price_items = price_data.get('item', [])
 57        price_text = price_items[0].get('finalPriceText', '') if price_items else ''
 58
 59        # Extract images from mapItemToProductTile
 60        tile = item.get('mapItemToProductTile', {})
 61        image_urls = tile.get('images', [])
 62
 63        # Download images
 64        for idx, img_url in enumerate(image_urls):
 65            self.download_image(img_url, product_id, idx)
 66
 67        # Extract reviews
 68        review = product.get('review', {})
 69
 70        return {
 71            'sku': product.get('partNumber'),
 72            'name': product.get('name'),
 73            'brand': product.get('brandName'),
 74            'manufacturer': product.get('manufacturerName'),
 75            'price': price_text,
 76            'uom': product.get('uom'),
 77            'specifications': specifications,
 78            'description': product.get('description', {}),
 79            'images': image_urls,
 80            'reviews': {
 81                'rating': review.get('rating'),
 82                'count': review.get('count'),
 83            },
 84        }
